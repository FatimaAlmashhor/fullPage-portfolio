Exhale

WEBSITE

Created a mental health site with a team using React, Tailwind CSS and Firebase database. This website has a total of 7 pages. They consist of a self assessment process to show a number https://exhale.netlify.app of questions to help diagnose the state of the user, reference to doctors and articles and a sign up and login designed through a pop-up.we designed the website UI using Figma. Tools used for this project were Circle Integrity, Airbnb, Pretter and Eslint, and Lottie for SVG animation. I worked with a team using GitHub and we used Netlify to publish the site.
