WEBSITE

Created an eCommerce website using HTML, JavaScript, CSS and PHP. The purpose of this specific website was to provide a platform for selling devices for the company. This website utilized databases through MYSQL. The website consists of an admin page to detect sold out items, add items, and to manage users and employees. The user can create a profile, add their items to a cart and can write comments for specific items
